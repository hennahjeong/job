비즈니스를 이해하는 데이터 분석가
Portfolio

마케팅·광고·디지털 도메인에서 데이터 분석과 비즈니스 인사이트 도출을 수행해왔습니다. 글로벌 광고 투자비 분석, 웹 분석(Adobe Analytics), SQL 기반 타겟 오디언스 분석 등 다양한 프로젝트 경험을 바탕으로, 데이터를 비즈니스 의사결정으로 연결하는 것을 강점으로 하고 있습니다.

핵심 프로젝트

Project 1. LG베스트샵 사분면 기반 오디언스 세분화

역할 주도 (기획 → 설계 → 보고서) 도구 Excel, PowerPoint 소속 아이지에이웍스 MI컨설팅팀

문제 정의

LG베스트샵(HS애드 경유)은 오프라인 거점 매장 반경 내 방문자/거주자/활동자의 특성을 파악하고, 각 매장에 최적화된 마케팅 전략 수립이 필요한 상황. 기존에는 오디언스를 하나의 덩어리로 보고 있어 매장별 차별화된 접근이 어려웠음.

데이터 수집 및 전처리

  • 자사 DMP 기반 오디언스 데이터 활용
  • 매장 반경 3km 내 거주자/방문자/활동자 데이터 수집
  • 데모그래픽, 관심사, 페르소나, 미디어 소비 행태 데이터 정제

분석 방법론

사분면(Quadrant) 기반 오디언스 세분화 프레임워크를 직접 설계했습니다.

  • 오디언스를 2개 축으로 나누어 4개 그룹으로 세분화
  • 각 그룹별 데모그래픽 특성, 관심사, 라이프스타일 패턴 분석
  • 지역 매장별 오디언스 그룹 밀집도 매핑

비즈니스 인사이트

  • 분석 프레임워크의 실효성이 입증되어 클라이언트로부터 추가 프로젝트 요청 발생
  • 이 프레임워크는 이후 아파트멘터리 프로젝트에서 스코어링 모델로 확장 적용됨

Project 2. 아파트멘터리 오디언스 세분화 + 스코어링 모델

역할 주도 (프레임워크 확장 + 스코어링) 도구 Excel, PowerPoint 소속 아이지에이웍스 MI컨설팅팀

문제 정의

인테리어 브랜드 아파트멘터리가 신규 오프라인 매장 입지를 선정하기 위해, 전국 단위에서 핵심 타겟 유저가 밀집한 지역을 데이터 기반으로 파악할 필요가 있었음.

분석 방법론

베스트샵에서 검증된 사분면 기반 세분화 프레임워크를 확장하고, 스코어링 모델을 추가 적용했습니다.

  • 자사 vs 경쟁사 사용자 특성 비교 분석
  • 지역별 핵심 유저 밀집도, 아파트 단지 특성, 상권 데이터를 종합하여 가망성 스코어 산출
  • 스코어 기반 지역 우선순위 도출

비즈니스 인사이트

  • 정성적 분류(세분화)에서 정량적 스코어링으로 분석 방법론이 진화
  • 데이터 기반 입지 선정이라는 구체적 비즈니스 의사결정에 직접 기여
  • 분석 프레임워크의 재사용성과 확장성을 실증

Project 3. 삼성닷컴 Adobe Analytics 기반 웹 분석

역할 직접 수행 도구 Adobe Analytics, Excel 소속 펑타이코리아 인사이트팀

문제 정의

삼성닷컴 글로벌 사이트의 캠페인 성과, 페이지 트래픽, 사용자 행동 패턴을 분석하고 전환율 개선 방향 도출 필요.

분석 방법론

다차원 웹 분석을 수행했습니다.

  • 페이지 트래픽 분석: 방문자, 순 방문자, 페이지 뷰, 체류시간, 이탈률
  • 전환 분석: 캠페인 페이지 유입 → 전환 → 구매 퍼널
  • 사용자 경로 분석: 주요 이탈 지점 파악, 전환 병목 구간 식별
  • 콘텐츠 효과 분석: CTA 버튼 반응률, 히트맵 기반 인터랙션 분석
  • A/B 테스트 분석: Cart Reminder A/B 테스트 분석 수행

비즈니스 인사이트

  • 데이터 기반 사용자 행동 분석 → UX/콘텐츠 개선 방향 제안
  • A/B 테스트를 통한 페이지 UI/UX 개선 근거 제공
  • 웹 분석 플랫폼(Adobe Analytics) 실무 활용 역량 확보

Project 4. DAI팀 타겟 오디언스 분석

역할 전 과정 담당 도구 SQL, Excel, PowerPoint 소속 아이지에이웍스 DAI(Data Analytics & Insight)팀

문제 정의

자회사(디지털 렙사/대행사)가 클라이언트 마케팅 비딩에 참여할 때, 데이터 기반 타겟 오디언스 분석이 필요. 자회사는 자체 데이터가 없어 본사 데이터 역량을 활용한 분석 보고서를 요청.

데이터 수집 (SQL 추출)

  • SQL을 사용하여 자사 DMP에서 데이터 직접 추출
  • 추출 항목: 데모그래픽, 관심사, 페르소나, 미디어 소비 행태
  • 자사 앱 사용자 및 경쟁사 앱 사용자 데이터 각각 추출

분석 방법론

자사 vs 경쟁사 비교 분석 프레임워크를 적용했습니다.

  • 자사와 경쟁사 사용자의 데모그래픽 비교
  • 유저들의 관심사, 페르소나, 자주 사용하는 미디어 분석
  • 경쟁사에서 유저를 데려올 전략 포인트 도출

비즈니스 인사이트

  • 데이터 기반 타겟 오디언스 전략 수립 → 자회사의 비딩 경쟁력 강화에 기여
  • SQL 추출 → 분석 → 인사이트 → 보고서 전 과정 수행 역량 확보
  • 다양한 산업군(금융, 유통, 식품, 패션, 게임 등) 분석 경험 축적
